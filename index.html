Vamos de dicas?

Não se desespere! Nem se ache burra ou incapaz (note que aqui antecipo sua reação, uma vez que você é exatamente assim).

A chave decimal é composta por 3 números de dois dígitos. Você deve convertê-los individualmente para números binários. Por fim, concatene os números binários em sequência.

Ex: 19, 05, 14

      
        19 => 10011
        05 => 101
        14 => 1110

O código será 100111011110.

Por onde começar?

É aqui que a coisa fica legal. Tenho três dicas importantes. Cada uma delas vai te levar a uma pessoa diferente. Uma pessoa próxima a você, tipo.. um parente. Essa pessoa possui a dica que vai te levar a um dos três números decimais.
Atenção! Se errar a sequência, erra a chave também. Descobrir a sequência correta faz parte do jogo.
